Use JavaScript to show or hide a password in an HTML form.

Source code found at: codepen link

Often, when entering a password, you need to ensure that the password is known and correct. There are several ways to accomplish this. First, you could require the client to enter the password into two separate fields, then compare the two values and indicate whether they match or not. Second, you could reveal the last typed character for a short duration before hiding it along with any other previously typed characters. Third, you could provide a means for showing and hiding the password, allowing the client to visually confirm the typed password. (00:04)

This short video demonstrates a simple JavaScript method for the third option . This example is stored on codepen.io for review and use, with the link provided in the description . On the left is a very simple form consisting of a label, a password input, and a span , which will be used as the toggle mechanism to reveal or hide the password. Because this is CodePen, the HTML, CSS, and JavaScript are linked by default. If this were a web page, the three documents would need to be linked using code in the form. Note the type of password and the id of 'pword' for the input, and the id of 'pswdButton' for the span, as these are important for the functionality. The CSS is very simple and will be hidden for the remainder of the video, with the focus on the JavaScript. (00:36)

Line 1 of the JavaScript is a variable that stores the reference to the span element in the HTML form. Line 2 establishes an event listener on the span. The event being listened for is <- a click, which is triggered when the mouse clicks on the target or if the enter or return key is pressed when the target has focus, usually by being tabbed to. If the click event occurs, the anonymous function (the second parameter in the event listener) is executed. (01:11)

Line 3 is a variable that stores the reference to the password input. Line 4 captures the value of the type attribute of the password input. By default, the type should be 'password'. Line 5 checks to see if the value is 'password'. If so, lines 6 and 7 are executed. Line 6 changes the value of the type attribute to 'text', removing the bullets in the input and revealing the text characters. Line 7 changes the text of the span from "show password" to "hide password". (01:54)

Line 8 is a simple else statement that is executed if the input type is anything other than 'password'. This causes lines 9 and 10 to execute. Line 9 changes the input type to 'password', replacing any readable text with bullets. Line 10 changes the text of the span from "hide password" to "show password". Line 11 ends the else statement. Line 12 closes both the anonymous function and the event listener. (02:37)

As you can see, the JavaScript is rather simple but allows the client to verify the password. This method could be used during a registration process as well as in any other situation where the client needs to see what would otherwise be hidden. (03:25)
